# apps

This directory is reserved for user-facing applications.

Typical examples:

- web frontends
- mobile clients
- desktop applications
- end-user command-line tools

## Current Status

No application projects have been added yet.

When a new app is created here, add a project-level `README.md` that explains:

- the user problem it solves
- how to run it locally
- how to build or deploy it
- any required configuration
